What is SubMiner?

SubMiner is a program that aims to find and download subtitles with the minimum possible effort. Upon installing, it will add two entries in your Windows context menu. This means you can right click your video file and ask SubMiner to download the subtitle for it. Simple as that.

Quick Start

  • Download and install SubMiner
  • Right click your video and select the option: "SubMiner: Download First"
  • The first found subtitle will be downloaded to the same directory and with the right name

SubMiner is not yet perfect, so although it works most of the time (for popular files), it may let you down from time to time. If you are a programmer and want to help make it better, please visit SubMiner's github page.
